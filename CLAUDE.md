# tcloud

TypeScript SDK wrapping the Tangle Router for inference, sandbox, and eval operations.

- Packages: `tcloud`, `tcloud-agent`, `tcloud-relayer`
- Talks to: `router.tangle.tools` via `/v1/*` and `/api/*`
- Credentials: `sk-tan-*` API keys (router-issued)
